# Reality Toolkit - Windows XR Platform Module

The Windows XR platform components for the [Reality Toolkit](https://github.com/realitycollective/com.realitytoolkit.core). This package enables your Reality Toolkit based project to run on Windows XR devices.

## What's included?

- Data Providers for the Camera System
- Data Providers for the Input System

## Supported Devices

- Microsoft HoloLens 2
- Windows Mixed Reality Headsets

## Requirements

- [RealityToolkit.Core](https://github.com/realitycollective/com.realitytoolkit.core)
- [Unity 2020.3 and above](https://unity.com/)

## Getting Started

### OpenUPM

The simplest way to getting started using the Windows XR platform package in your project is via OpenUPM. Visit [OpenUPM](https://openupm.com/docs/) to learn more about it. Once you have the OpenUPM CLI set up use the following command to add the package to your project:

`openupm add com.realitytoolkit.windows.xr`
